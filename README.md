# Cowin-API
A JAVA SWING APPLICATION which get you INFORMATION About VACCINE AND HOSPITALS on basis of CITY , DOSE and DATE . Also allow the user to DOWNLOAD VACCINE CERTIFICATE. All done by using CO-WIN PUBLIC API.
